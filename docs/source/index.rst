EduSim — Un système de simulation pour la formation universitaire
=================================================================
**EduSim** est une bibliothèque Python permettant de simuler un environnement
informationnel pour soutenir l'apprentissage de la valorisation des données
en contexte professionnel. Cette version offre un premier environnement de
simulation dédié au milieu scolaire primaire québécois.

.. note::
   Cette version est la première ébauche du simulateur.
